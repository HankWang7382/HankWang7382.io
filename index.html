Clustering Stability Analysis of the Gender Equality Evaluation Model

TIP Taiwan Gender Equality Index · Hierarchical Clustering

Investigated the stability of hierarchical clustering results underlying the Gender Equality Evaluation Model, which groups listed and OTC companies by female workforce structure. Evaluated five linkage methods (Single, Complete, Average, Centroid, Ward's) and three cluster validity indices (Silhouette, Davies-Bouldin, Dunn Index) across annual rebalancing cycles from 2016 to 2021. F-measure via confusion matrix was used to quantify year-over-year clustering consistency. Optimal configuration identified as Ward's method with Dunn Index, cluster count constrained to 4–9, and IQR-based outlier detection, achieving the highest adjusted F-measure and Kendall's tau stability.
